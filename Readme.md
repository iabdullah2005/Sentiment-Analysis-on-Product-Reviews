# Sentiment Analysis on Amazon Product Reviews

## Overview

This project performs sentiment analysis on Amazon product reviews using Natural Language Processing (NLP) and Machine Learning techniques.

## Objectives

* Classify reviews into Positive, Neutral, and Negative sentiments.
* Apply text preprocessing techniques.
* Extract features using TF-IDF.
* Compare machine learning models.
* Evaluate model performance.

## Dataset

Amazon Fine Food Reviews Dataset

## Text Processing

* Lowercasing
* Tokenization
* Stopword Removal
* Text Cleaning

## Feature Engineering

* TF-IDF Vectorization
* Unigrams and Bigrams

## Models Used

### Multinomial Naive Bayes

Accuracy: 81.64%

### Linear Support Vector Machine (SVM)

Accuracy: 86.06%

## Results

The Linear SVM model outperformed Naive Bayes and was selected as the final sentiment classification model.
